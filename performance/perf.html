# 性能

从两方面思考 web 的性能问题：页面纬度、运行纬度

  • **页面纬度：**页面的加载所经历的过程，思考在这个过程中有哪些是需要重点关注并能做一些优化的事情。
  • **运行维度：**通常指动画(js动画、css动画、其他动画等等)，以及交互性能问题等。
